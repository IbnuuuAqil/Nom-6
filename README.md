Fungsi Program
Program ini dibuat untuk memudahkan pengguna dalam menjumlahkan dua buah matriks. Daripada menghitung elemen matriks satu per satu secara manual—yang bisa cukup merepotkan—program ini akan melakukannya secara otomatis. 

Cara Kerja Program
Cara kerja programnya sederhana. Pertama, pengguna diminta memasukkan ukuran matriks, seperti jumlah baris dan kolom. Setelah itu, pengguna akan mengisi nilai untuk setiap elemen di kedua matriks. Program lalu menjumlahkan setiap elemen dari matriks pertama dengan elemen yang posisinya sama di matriks kedua. Jika semua proses sudah selesai, program akan menampilkan hasil penjumlahannya dalam bentuk matriks baru. Jadi, pengguna langsung bisa melihat hasilnya tanpa harus menghitung manual.
